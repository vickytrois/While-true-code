Reproduzindo áudios

Para reproduzir áudios num site, usamos a marca <audio>.

Caso o atributo autoplay não funcione, nós podemos utilizar a marca <audio> de uma forma um pouco diferente:

Alguns navegadores não tem compatibilidade com arquivos mp3, por exemplo. Uma alternativa são os formatos ogg e wav.

Os códigos de mídia para esses formatos são audio/ogg e audio/wav (não é aconselhado o uso de arquivos .wav, pois eles são muito pesados).

Para mp3 usamos audio/mpeg.
