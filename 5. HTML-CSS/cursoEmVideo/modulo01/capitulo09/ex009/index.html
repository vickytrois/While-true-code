Trabalhando com Listas

Aaaaah chegou a horaa! Amo listas, partiu ver como elas funcionam em HTML!

Para listas ordenadas usamos a marca mark <li> e para não ordenadas, a marca <ul>.

Listas Ordenadas

  1. Item
  2. Item
  3. Item
  4. Item

Listas não-ordenadas

  • Item
  • Item
  • Item
  • Item

Fechar a marca <li> se tornou opcional a partir da quinta versão do HTML.

Personalizando listas

Por padrão, listas ordenadas são organizadas por números inteiros e começam em 1.

Se precisarmos de ordens diferentes dessa, usamos os parâmetros "type" e "start" para configurá-las.

O parâmetro type

Nas listas ordenadas, temos como opções de tipos de ordenação os tipos: 1, A, a, I e i.

Já os marcadores possíveis para as listas não-ordenadas são disc, circle e square.

O parâmetro start

Quando precisarmos mudar o valor inicial de uma lista ordenada, digitamos: 

      <ol> start="<valor inicial desejado>">
        <li>Item
        <li>Item
        <li>Item
        <li>Item
      </ol>

Junção entre listas

Da mesma maneira que podemos aninhar condições, podemos aninhar listas: 

      <ol>
        <li>Lista1
          <ol>
            <li>item1Lista1
            <li>item2Lista1
            <li>item3Lista1
          </ol>

        <li>Lista2
        <ol>
          <li>item1Lista2
          <li>item2Lista2
          <li>item3Lista2
        </ol>
      <ol>

Listas de definições

Uma ferramenta bastante útil para criação de wiki (opinião pessoal). Usamos a marca <dl> para a estrutura da lista e as marcas <dt> e <dd> para os termos e as definições, respectivamente.

Ex.:

HTML
Linguagem de marcação com foco no conteúdo de sites. Abreviação de HyperText Markup Language.
CSS
Linguagem de folha de estilo com foco no design de um site. Abreviação de Cascading Style Sheet.
JS
Linguagem de programação que foca na interação entre site e usuário. Abreviação de JavaScript.
