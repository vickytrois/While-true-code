Estudos sobre DOM

O que é DOM?

DOM significa Document Object Model (modelo de objetos para documentos). Um conjunto de objetos dentro do navegador que dá acesso aos componentes internos de um site.

Árvores DOM

Uma árvore DOM começa da raiz, window, que possui diversos objetos dentro de si. Alguns exemplos são: location, document e history.

Dentro de document, um objeto importante é o html, que por sua vez, tem como objetos filhos head e body.

Navegando dentre os elementos da árvore DOM

Há 5 métodos de seleção desses elementos:

  • por Marca
  • por ID
  • por Nome
  • por Classe
  • por Seletor

    • Por Marca:

    getElementsByTagName();

    Nesse exemplo, usamos dois parâmetros. Um vai entre () e o outro entre []. Entre parênteses colocamos o nome da marca que usaremos para buscar os elementos, neste caso p. Já nos colchetes, colocaremos a posição delas no corpo do documento HTML iniciando por zero. Se quizermos acessar a primeira marca de parágrafo usamos dentro dos colchetes 0, se quizermos o segundo, usamos 1 e assim por diante (mesma lógica que se usa para índices na maioria das linguagens de programação).

    Por ID:

    getElementById();

    Para acessar um elemento por sua ID, primeiro precisamos criar uma ID para ele. Dentro dos colchetes angulares, escrevemos "id = " e atribuimos o valor desejado. Como por exemplo quando tornamos o fundo de "getElementById" da cor "aqua":

    Por Nome:

    getElementsByName()[]

    Acessar um elemento pelo seu nome é parecido com acessar um documento pela sua ID. No entanto, ao invés de termos dentro da marca "id=<valorID>", utilizamos apenas "=<nomeMarca>".

    Já que o comando seletor tem a palavra elementS, no plural, deveremos informar qual a posição ocupada pelo elemento desejado.

    Por Classe:

    getElementsByClassName()[]

    Para acessar um elemento pela sua classe é muito parecido com quando o acessamos pelo seu nome e ID. Para isso, ao invés de introduzirmos na sua marca "id=" ou "name=", basta introduzirmos "class=".

    Por Seletor:

    querySelector()
    querySelectorAll()

    Selecionar elementos por seletor é uma técnica bastante recomendada pelas documentações existentes. Podemos usar como referência IDs ou classes.

    Considerando a sintaxe do CSS, IDs são representadas por '#' e classes por '.' .
