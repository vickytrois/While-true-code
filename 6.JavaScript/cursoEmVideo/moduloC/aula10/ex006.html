Eventos DOM

Eventos são o que tornam possível que usuários interajam com dispositivos. Quando colocamos o mouse sobre uma certa parte da tela e recebemos uma "resposta".

Eventos DOM são muito diversos e é bastante difícil saber todos de cor, por issó é importante ter boas referências de pesquisa quando houver necessidade. Duas boas referências são:
Lista de eventos DOM da W3schools e
Lista de eventos DOM da Mozilla

Tipos de eventos DOM

Alguns tipos de eventos de JS são "animation", relacionados à API "Web Animation", "Clipboard", relacionados à API "Clipbpard", entre muitos outros.

Um tipo muito importante e o que vamos fazer alguns testes aqui são os eventos do tipo "Mouse".

Eventos de mouse

Dentre os eventos de mouse podemos citar "mouseenter", "mousemove", "mousedown", "mouseup", "click" e "mouseout".

Tratando eventos

Para dispararmos eventos, usamos funções. Um conjunto de linhas executadas apenas quando um evento ocorrer. Em JS uma função é declarada com a seguinte sintaxe:

function <nomeFuncao>(parâmetro){
 instruções
}
